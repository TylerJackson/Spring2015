Starbucks Card is an app that allows users to buy a drink from Starbucks
using their gold card! It is preloaded with $40.00.

When you enter the store (aka run the app) you can choose to buy a drink or exit.

After purchasing a drink or exiting, the app will tell you your final balance
and the number of drinks you purchansed.

---------------------------------------------------------------------------------

Use the following command to compile:
g++ -g -fno-stack-protector -o starbucks

Use the following command to run:
./starbucks

---------------------------------------------------------------------------------

A successful attack will allow the user to purchase a drink without spending any of
the money on his/her gold card.
